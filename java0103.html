범위
byte형 -2^7 ~ 2^7 -1 / -128 ~ 127
short형 -2^15 ~ 2^15 -1 / -32768 ~ 32767
int형 -2^31 ~ 2^31 -1 / -21억 ~ 21억사이
long형 - 2^63 ~ 2^63 -1 / 그냥 대충 22억 넘으면 L 붙여라

고정소수점 데이터 표현 방식 = 정수 표현에 사용

부동소수점 데이터 표현 방식 = 실수 표현에 사용

=> 지수 표현 방식을 사용하여 수치 데이터를 표현

+- 가수부 * 10^ 지수부

0.00000000001234 > 0.1234

0.1234 * 10^-11 하면

정밀도에 있어서 double 이 float 보다 정밀하다.

결론 : 실수형 데이터 타입은 정수형 데이터 타입보다 큰 타입으로 취급됨.
